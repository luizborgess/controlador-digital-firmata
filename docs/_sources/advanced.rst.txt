**Recursos Avançados**
===============================================

Alguns recursos extras estão incluídos no software dentro da pasta útil, nesta contem scripts em Python, para
auxiliar alguns recursos avançados no software, são eles create_csv.py e plot_csv.py.

Create_csv.py
---------------------

Este arquivo permite que se crie um perfil de distúrbio para aplicar no software pela opção carregar csv, esta
opção irá carregar o arquivo csv, e plotar e aplicar no sistema como um distúrbio dentro do intervalo selecionado na
criação do mesmo.


Plot_csv.py
---------------------

Como o software possui a opção de salvar todos os dados adquiridos durante um experimento em um arquivo csv, ele também
possui um script para plota-los em um gráfico posteriormente se desejado, basta inserir a localização do arquivo csv
salvo que o script irá plotar o csv em um gráfico para melhor visualização.
